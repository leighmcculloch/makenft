# makenft

A utility website for creating NFTs on the [Stellar] network.

Deployed at [leighmcculloch.github.io/makenft].

## Development

### Install dependencies

```
npm install
```

### Run locally

```
npm run dev
```

### Build for deployment

```
npm run build
```

### Deploy

```
npm run deploy
```
