An NFT will be created on the Stellar network. If a file is uploaded it will be permanently stored on the Stellar network. It will not be possible to delete the file from the network.

Your account will pay the issuing account reserve and all transactions fees. The reserve will not be reclaimable.

Once the NFT is created you can send or sell your NFT using your wallet, or DEX app.
